FiveByFive

Game #1

Drag and drop tiles onto the white squares of the grid to form 6 words (3 across, 3 down). There might be multiple valid solutions; you only need to find one.

The "HINT" button (press and hold) will show you progressively more expected letters, but you only get 3 hints maximum.

When the game's over, the "CHECK" button (press and hold) will show you the expected solution.

NOTE: The game verifies words against the list of 5-letter words that are valid for typical word games. The expected solution only uses the (roughly) 500 most common words, but you can use any of them.
